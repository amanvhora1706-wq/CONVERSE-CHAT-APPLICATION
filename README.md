## Converse – Secure Chat App (UI/UX Prototype)
Converse is an internal communication platform prototype co-designed for the ADIT Placement Department during my internship at Tech Elecon Pvt. Ltd. It focuses on clean navigation, high-fidelity interactions, and modern user workflows.

### 🚀 Key FeaturesCore 
#### Communication: 
Interactive screens designed for individual messaging, group chats, and media sharing.  
#### Integrated Call UI: 
Dedicated layouts for seamless voice and video calling streams.
#### Advanced Security Concepts: 
Mockups for biometric authentication including Face ID, fingerprint scanning, and secure PIN setups.

### 🛠️ Design & Architecture Tools
1.UI/UX Design Platform:Figma (used for creating responsive wireframes, reusable UI components, and dynamic interaction prototypes).
2.System Flow Mapping: Structured system analysis visualized via custom Use Case, Activity, and Sequence diagrams.

### 💼 My Responsibilities
-> User Flow Mapping: Designed end-to-end user journeys from onboarding to active messaging states.  
-> Security & UI Layouts: Created conceptual layouts for authentication and multi-layered security settings.  
-> Visual Styling: Built consistent, component-based visual assets ensuring smooth, intuitive navigation.

### ⚠️ Project Scope & Limitations
Design Prototype Focus: This repository reflects the high-fidelity UI/UX prototype created during the internship timeline to showcase visual architecture and user navigation; it is not a fully compiled Android package.  

Target Environment: Custom-designed specifically around the communication flows required for institutional placement departments rather than generalized commercial use.
